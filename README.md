# Solana Spot Trading Bot

Python-based systematic trading bot for Solana spot markets using trend-following and momentum signals on 4h candles.

## Highlights
- VPS deployment
- Live paper-trading logs
- Historical backtest on ~7 years of Binance data
- Strategy based on EMA trend filter + RSI momentum confirmation

## Project Structure
- `index.html`: GitHub Pages presentation
- `data/`: live logs
- `bot/`: code snippets
- `equity_curve.png`: backtest result chart
